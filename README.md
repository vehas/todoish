# install shadow-cljs and yarn
```
npm install -g shadow-cljs
npm install -g yarn
```

----------------
# Starter Template from https://github.com/shadow-cljs/examples

Simple Shadow CLJS Template Project

## Getting Started

```
yarn watch
open http://localhost:8280
```

## Testing

Interactive:

```
yarn watch-test
open http://localhost:8290
```

CI Server:

```
yarn test
```

cider-connect
